## Context

当前 `research/` 目录下存放了五篇深入的同人展（Comiket 和 Comicup）学术分析报告。这些报告广泛应用了统计学、经济学、地理学等领域的专业计算（如 DBI/LQ 指数、Moran's I 自相关、卡方检验与 Cramér's V 效应量、SDI 供需比等）。当前存在的问题是，这些学术性的概念对于普通动漫同人爱好者（如普通的摊主、买家、读者）阅读门槛极高。

本设计旨在提供一套系统化的重构策略，使得这五篇研究报告能够在保留严谨数学表达和准确数据的前提下，转型为对普罗大众友好的“科普向”同人圈深度报告。

## Goals / Non-Goals

**Goals:**
1. 保留原本报告中全部 LaTeX 格式的数学公式，不损失学术和技术严谨性。
2. 每一个核心公式和统计检验章节都通过“四层渐进式”（学术描述 $\rightarrow$ LaTeX 公式 $\rightarrow$ 变量大白话拆解 $\rightarrow$ 宅文化通俗翻译）的形式重构。
3. 替换和映射所有的枯燥统计学词汇，将它们类比为同人展上的日常体验。
4. 保证原有的数据完全一致，不篡改任何计算出的统计数字与结论。

**Non-Goals:**
1. 不涉及任何 Python 代码或 SQL 查询文件的逻辑修改（即不改动 `src/` 或 `research/scripts/`，仅对报告文档进行文风和组织上的优化）。
2. 不删减任何原有的局限性声明（如因果推断限制、循环论证警告等），以维持报告的学术反思深度。
3. 不更改报告已定义的图片或 Mermaid 拓扑结构。

## Decisions

### 决策 1: 采用“四层渐进式”结构组织数学章节
- **选择**: 在公式呈现之后，强制紧跟大白话的变量释义和富有二次元色彩的同好日常类比。
- **原因**: 直接删除公式会失去专业报告的价值，而仅仅用文字描述公式又过于抽象。四层结构能完美平衡两端：学术读者直接看公式，普通读者跳过公式看“宅文化翻译”。
- **替代方案**:
  - *方案 A*: 仅提供文字解释，将公式移到附录。缺点是破坏了正文的连贯性，学术审查时体验差。
  - *方案 B*: 采用双版本报告（学术版与科普版）。缺点是维护成本翻倍，且容易产生内容分化。

### 决策 2: 统一术语的趣味同人化类比字典
- **选择**: 制定统一的词汇对照字典，保证五篇报告中的比喻和叫法一致。
  - Moran's I $\rightarrow$ “摊位抱团指数（或街区抱团温度计）”
  - SDI $\rightarrow$ “秒空指数（或排队温度计）”
  - DBI / LQ $\rightarrow$ “同人狂热指数（或创作偏离度）”
  - CR10 $\rightarrow$ “题材霸屏率（或流量分流集中度）”
  - 卡方检验与 Cramér's V $\rightarrow$ “因果八卦验证与标签捆绑度”
- **原因**: 保持五篇文档的文笔风格和核心隐喻高度对齐，形成一个完整的研究报告宇宙。

## Risks / Trade-offs

- **[Risk]** 通俗化类比可能在学术上不够精确，导致过度简化。
  - *Mitigation*: 在通俗化翻译的前后，依然严格保留原有的学术表述和局限性声明（例如，在解释 SDI 时保留生产时滞和物理筛选机制的分析，在解释 Moran's I 时保留关于官方规制循环论证的学术批判）。
- **[Risk]** LaTeX 公式在 Markdown 渲染中可能由于文档编辑产生格式错位。
  - *Mitigation*: 严格遵守标准的 Markdown/MathJax $...$ 和 $$...$$ 语法，编辑后仔细核对，确保公式的物理结构不乱。
