## ADDED Requirements

### Requirement: 保留学术严谨性与LaTeX数学公式
系统或报告编辑者在优化报告文本时，MUST 保留原本报告中存在的所有 LaTeX 格式的数学公式（如集中度 $CR_n$、区位商/同人偏离度 $LQ$ 与 $DBI$、空间自相关莫兰指数 $Moran's I$、卡方独立性检验 $\chi^2$、平面坐标映射与步行距离估算等公式）。

#### Scenario: 成功保留所有数学公式
- **WHEN** 检查重构后的五个研究报告 markdown 文件时
- **THEN** 所有数学公式仍然以 LaTeX 形式完整保留在文件中

### Requirement: 采用“四层渐进式”结构重构数学公式与学术分析章节
对所有引入数学公式或深度统计分析的章节，MUST 采用以下四层渐进式结构进行展示和解释：
1. 学术陈述 (Academic Statement)：保留原有的严谨学术表述；
2. LaTeX 数学公式 (LaTeX Formula)：给出严格的数学公式；
3. 变量拆解 (Variable Breakdown)：用大白话逐个解释公式中的所有变量；
4. 宅文化翻译 (Otaku Translation)：用通俗的二次元、同人展大白话/比喻解释公式背后的实际物理含义。

#### Scenario: 成功重构核心章节
- **WHEN** 用户阅读任意一篇重构后的报告中的公式解析部分时
- **THEN** 能够看到学术陈述、LaTeX 公式、大白话变量拆解以及宅文化生动翻译这四个层次的内容

### Requirement: 引入通俗术语映射
报告中的统计学与计量分析术语，MUST 进行通俗化的比喻或类比翻译：
- 同人偏离度 (DBI / LQ) $\rightarrow$ 映射为“同人狂热指数/创作偏离温度计”；
- 莫兰指数 (Moran's I) $\rightarrow$ 映射为“摊位抱团指数/街区抱团温度计”；
- 供需偏离度 (SDI) $\rightarrow$ 映射为“秒空指数/排队温度计”；
- 行业集中度 (CR5 / CR10) $\rightarrow$ 映射为“题材霸屏率/流量分流集中度”；
- 卡方独立性检验 ($\chi^2$) 与 Cramér's V 效应量 $\rightarrow$ 映射为“因果八卦验证/标签捆绑度”；
- 空间自相关与空间邻接矩阵 $\rightarrow$ 映射为“邻近街区是否都在卖同人本/摊位串门指数”。

#### Scenario: 术语映射在行文中展现
- **WHEN** 全文完成重构后
- **THEN** 读者能够看到这些术语对应的趣味同人圈类比与直白释义，帮助普通人无障碍理解

### Requirement: 保留原有数据结论与图表链接
报告重构 MUST 保持原有数据分析结果的准确性（不得更改任何统计数字，如 22,856 条数据、74.27% 绑定率、Moran's I 的 0.4482 等），且 MUST 保留原有的 mermaid 流程图、示意图及静态 PNG 图表路径链接。

#### Scenario: 校验数字与图表的完整性
- **WHEN** 校验重构后的五个 markdown 报告时
- **THEN** 报告中的所有具体数值与原始文件完全一致，且所有 mermaid 图表和图片链接完好无损
